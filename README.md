# Usage

Implementing wfsnav.js into your project is easy.
<br>

Add script right before closing ``</body>`` tag:

```html
<script src="https://your_domain.com/js/wfsnav.js"></script>
<script>
    const WebNavGoto = new webFullsetNavGoto({
        conteiner: '#header'
    })
</script>
```
